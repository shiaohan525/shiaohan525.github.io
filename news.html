滾鍍(免掛具)電著(泳)塗裝之進展 2005/9/1(Bulk Electro-coating)

1.簡介

一般機械五金零件、電子零件如果体積不大，構造堅固易於滾動的時候，在需要作電鍍表面處理的時候，滾筒電鍍 (Barrel Plating)即滾鍍是最佳的選擇，不必像吊鍍一個一個的以人工掛上吊架，人工貴每工時產量又低。 對這些相同的小零件如果想用塗裝的時候，吊掛噴漆是可保証品質均一但是單一零件的塗裝成本太高了，雖然吊掛噴塗塗裝成本相對於吊掛電鍍成本仍低唯塗裝或電鍍功能不同，依客戶之需求而定。 塗裝滾塗的主要問題在於塗料在塗裝後烘烤前有黏性的困擾 這是與電鍍最大的不同，如何像電鍍一樣有滾鍍的生產線一直是塗裝界思考的問題，如（圖一）所示是以各色兩液型塗料一面薄噴一面烘烤 ( 約 60 – 80 度 C ) ，可噴出多種顏色配合各類金屬鈕扣衣服顏色是其優點，此法的品質缺點為光澤度低 表面因互相磨擦產生毛毛的感覺，浸旋 ( Dip / Spin ) 塗裝也是一種塗裝之方式 但用於黃銅電鍍後薄塗一層透明噴漆為多。

因此滾鍍電著方式約於七O年代即應運而生，此以水性帶有正或負離子樹脂電著( 電泳 ) 塗裝方法從60 年代開始以陽極電泳大量使用於汽車及其零件、五金配件等當然以輸送帶吊掛為主， 美國 Armco Steel Corp. 從70 年代即陸續發表多種滾鍍電著專利。
此後對於五金小零件，電子零件如何用電泳大降成本，且品質可以媲美弔掛電著， 正是各界努力的目標，例 圖二、 三 即為免直接掛具 吊籃式電泳塗裝線，由台灣宗洋公司約10年前所設計現仍使用中。
2.用途

凡是零件的幾何形狀多加工後不糾纏在一起的有凹陷或突脊小零件都是合適的電泳的大宗零件，以單位面積的重量來說，面積大重量輕的小零件 ( 如螺絲有凹凸單位面積重量輕 ) 客戶允許有些接觸點 ( Touch Mark )而邊緣防銹 ( Edge Coverage )品質要求普通的，就是適合的電著零件，目前以螺絲 ( 如木工螺絲 汽車螺絲 ) 例如 (圖四)、管夾、支撐架等 。

3.技術及專利大概

1984 年 電著塗裝技術先驅 Dr. George Brewer 即發表於 Bulk Electro-coating 於 Electrocoat Conference Proceedings.下文即是以此為本發表於台灣塗料與塗裝雜誌有關免直接掛具之大宗零件電泳塗裝設備。

以上之專利資料或許偏舊，新的專利仍不斷出現，基本上仍以此為本，但以滾筒狀的電泳方式較佔優勢因為產量大，每個滾筒可放上百或數百公斤如螺絲之零件，如下圖五 為其中一例，新穎專利設計推陳出新、操作更方便。

4.相關之電著塗料

早在七O年代陽極電泳盛行之年代即有滾鍍電泳，如美國Valspar公司的VectrpCoat 400 Black 即是符合美國Chrysler 車用螺絲之陽極電泳漆最近還是有人使用， 一般而言 環氧樹脂電泳塗料在經電泳水洗後即可指觸不黏手，隨著電泳塗料之進步，新型之耐UV紫外線之陰極丙烯酸 ( Acrylic)樹脂系列及特性更佳之陰極環氧樹脂 (Epoxy電泳漆如美國 PPG Powercron AL ( Acrylic )系列 及 Powercron XL (Epoxy) 黑色系列，都已經過美國三大汽車廠的品質確認 ( 包含處理製程 )，但基本上丙烯酸系列較黏效果較差。

5. 結論

由於適合滾鍍五金小零件日漸增多，電子小零件量亦可觀，此種可節省成本之免直接掛具滾鍍方法及其電泳塗料，值得業界在設備及塗料上配合研究擴大市場。
